== Installation ==
This section describes how to install the WPForms Widget For Elementor Page Builder Plugins for WordPress get it working.

= 1) Install =

= Install: =
1. Go to the WordPress Dashboard "Add New Plugin" section.
2. Search For "WPForms Widget For Elementor Page Builder".
3. Install, then Activate it.

= OR: =
1. Unzip (if it is zipped) and Upload `ht-wpform` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
